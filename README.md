# Junction — Keep The Main Thing

A single-file HTML presentation for The Junction Church: *Keep The Main Thing The Main Thing*.

**Live site:** https://Dialled-In-Dads.github.io/Preach-Slides-Beta/

## What is this?

A zero-dependency, self-contained HTML slide deck. No build step, no npm, no server — just open `index.html` in any browser and it works.

## Want your own editable copy?

You can fork this repo and host your own version in about 10 minutes:

1. Click **Fork** (top right of this page) — this copies the repo to your own GitHub account.
2. In your fork, go to **Settings → Pages**.
3. Under *Source*, select **Deploy from a branch** and pick `main` / `/ (root)`. Save.
4. Wait a minute, then visit `https://<your-username>.github.io/Preach-Slides-Beta/` — your live copy.
5. Edit `index.html` directly in the GitHub web editor (or clone locally) — changes deploy automatically on push.

## Editing locally

Clone, open `index.html` in your browser, edit, commit, push. That's it.

```bash
git clone https://github.com/<your-username>/Preach-Slides-Beta.git
cd Preach-Slides-Beta
# edit index.html in any editor
git add index.html
git commit -m "Update slides"
git push
```
